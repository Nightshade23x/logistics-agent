# Current System Snapshot

## Project Status

The backend is now local-demo ready.

The system can accept a shopping/import request, route it through the User Agent, call specialist agents, prepare partner-review requests, and produce frontend-ready output.

Live partner Risk, Compliance, Trader, and Finance services are not connected yet. Their adapters and request contracts are prepared, but they currently return not-configured style responses.

## Completed Agents and Services

### User Agent

The User Agent is the orchestrator.

It detects request intent and routes the request to the correct specialist flow:

- Shopping request
- Logistics request
- Document request
- Text request fallback
- JSON request
- Document-file request

It also combines specialist responses and attaches final verdict, booking readiness, executive summary, and frontend sections.

### Shopping Agent

The Shopping Agent handles supplier and procurement planning.

Current capabilities:

- Parses shopping/procurement requests
- Selects supplier options
- Produces selected items
- Estimates procurement cost
- Creates supplier shortlist
- Creates purchase-order style output
- Produces handoff payload for Logistics Agent

### Logistics Agent

The Logistics Agent handles shipment planning.

Current capabilities:

- Calculates total CBM
- Calculates total weight
- Recommends container type
- Recommends FCL/LCL direction
- Detects fragile, heavy, non-stackable, and possible battery cargo
- Produces packaging, loading, and handling recommendations
- Produces logistics handoff payload for partner review

### Document AI Agent

The Document AI Agent handles document extraction and document consistency checks.

Current capabilities:

- Reads invoice-style input
- Reads packing-list style input
- Extracts shipment items
- Checks invoice and packing-list consistency
- Produces document handoff payload for Logistics Agent

### Partner Review Service

The Partner Review Service prepares calls for external partner agents.

Prepared partner agents:

- Risk Agent
- Compliance Agent
- Trader Agent
- Finance Agent

Current status:

- Partner request builder works
- Payload validator works
- Adapter skeletons work
- Live partner services are not configured yet
